许可证 ​

Lyra Code 采用双许可模式。

开源许可 ​

Lyra Code 原始源代码遵循以下协议发布：

GNU Affero General Public License v3.0 或更高版本（AGPLv3-or-later） ​

查看完整协议文本

AGPLv3 要点：

  • ✅ 可自由使用、修改和分发
  • ✅ 必须公开源代码
  • ✅ 修改版本同样必须以 AGPLv3 许可
  • ✅ 网络使用视同分发（网络传播也受 Copyleft 约束）

商业许可 ​

以下场景需要获取商业许可：

  • 闭源分发 Lyra Code 或其衍生作品
  • 私有修改而不公开源代码
  • 商业场景下免除 AGPL Copyleft 义务
  • 任何你不想遵守 AGPL 条款的使用场景

商业许可咨询请联系仓库作者。
