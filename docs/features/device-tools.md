# 设备与系统

Lyra Code 提供全面的设备诊断和系统交互能力。

## 设备信息

设备信息页面提供完整的硬件与软件概览：

### 硬件

| 类别 | 详情 |
|------|------|
| 制造商与型号 | 设备品牌和型号标识 |
| Android 版本 | 操作系统版本和 API Level |
| CPU / 硬件 | 处理器型号和硬件平台 |
| ABI | 支持的二进制接口（arm64-v8a 等） |
| 内存 | 总内存与可用内存 |
| 存储 | 内部存储容量与使用情况 |
| 显示屏 | 分辨率、像素密度、刷新率 |
| 网络 | 连接类型、运营商信息 |
| 蓝牙 | 适配器状态与能力 |
| 电池 | 电量、健康状态、充电状态、温度 |

## 硬件检测 Agent

Agent 可执行硬件检测，用于：

- **设备诊断** — 识别硬件问题和异常
- **故障排查** — 获取 AI 辅助的设备问题指导
- **配置对比** — 选购设备时比较规格

## 已安装应用识别

扫描和分析已安装的应用程序：

| 功能 | 说明 |
|------|------|
| 应用列表 | 浏览全部、用户或系统应用 |
| 包名 | 识别包名和版本 |
| APK 大小 | 查看安装占用空间 |
| 签名证书 | SHA-256 证书验证 |
| 搜索 | 按名称或包名查找应用 |
| 分页 | 高效处理大量应用列表 |

::: tip 安全
应用识别通过签名证书验证，帮助发现可疑或未经授权的应用。
:::

## Shizuku 与 Root 工具

面向高级用户，Lyra Code 提供系统级工具：

### Shizuku Shell

- 以 Android shell 用户身份执行命令
- `pm` 命令进行包管理
- `cmd` 和 `dumpsys` 进行系统诊断
- 以 shell 权限读取 `/data` 路径

### Root 命令

- 可配置的 `su` 命令
- 权限回退链
- 执行前完整命令预览

::: danger 重要提示
系统级工具可在 Agent 设置页面**独立禁用**。Shizuku 和 Root 工具均需要**逐项用户明确批准**。
:::

## 主题与显示

Lyra Code 提供丰富的视觉定制选项：

### 主题模式

- **浅色** — 清晰明亮的界面
- **深色** — 护眼深色模式
- **跟随系统** — 跟随设备主题设置

### Material You

可选的动态取色支持，从壁纸中提取色彩，不改变应用布局。

### 字号与刷新率

- 可调节字号，实时预览效果
- 跟随系统字号缩放
- 刷新率偏好：30 Hz、60 Hz、90 Hz、120 Hz
- 智能模式，跟随系统显示策略
