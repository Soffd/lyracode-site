## 什么是 Lyra Code？

Lyra Code 是一款 **Android 优先的本地 AI Agent 应用**。它将多模型聊天、文件操作、原生下载、命令执行、网页搜索、MCP、SSH、WebDAV、备份恢复、技能包（Skills）、设备诊断和用量统计等功能整合到移动端工作流中，让你的 Android 设备能够胜任编程、写作、研究、远程运维和自动化等各类任务。

::: tip 核心亮点
- 🔌 **多 AI 服务商** — 支持 OpenAI、Anthropic、Gemini 及自定义 API
- 🛠️ **丰富 Agent 工具集** — 文件、命令、网页、MCP、SSH、WebDAV
- 📦 **技能包系统** — 可导入知识包，应对专业场景
- 🎭 **沉浸式角色扮演** — 好感度系统 + 表情贴纸
- 📊 **使用统计** — Token 计数、会话统计、历史回顾
- 🌗 **主题与显示** — 浅色/深色、Material You、可调字号
:::
