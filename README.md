# Lyra Code 官网

> 用 Lyra Code 本身来制作 Lyra Code 的介绍网站 —— 没有比这更酷的方法了。

这是 Lyra Code Android 应用的官方介绍网站，使用 VitePress 构建，本仓库为构建产物，通过 GitHub Pages 部署。

访问地址：**[lyracode.yukisoffd.com](https://lyracode.yukisoffd.com)**

## 技术栈

- [VitePress](https://vitepress.dev/) — 静态站点生成器
- GitHub Pages — 托管与部署

## 本地构建

源码位于主仓库 [Soffd/Lyra-Code](https://github.com/Soffd/Lyra-Code)。
